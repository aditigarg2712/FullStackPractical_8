Introduction of HTML

The HyperText Markup Language,or HTML is the standard markup language for documents designed to be displayed in a web browser.It can be assisted by technologies such as cascading style sheets and scripting languages such as Javascript.Web browser receive HTML documents from a web server or from local storage and render the documents into multimedia web page.HTML describes the structure of a web Web-Page sementically and orginally included cues for the appearence of the document.
